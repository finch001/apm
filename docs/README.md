# 特性

-   自动捕获未处理的错误；
-   能够捕获 3 种不同的前端错误：JavaScript 执行错误，资源加载错误和 HTTP 请求错误。
-   支持通过 Source Map 还原出错源代码
-   记录出错前的鼠标点击、HTTP 请求、页面跳转、console 打印等用户行为，帮助您复现 BUG
-   支持收集 try/catch 捕获的错误；
