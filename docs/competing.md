# 前端监控

如何第一时间发现前端异常
如何快速及时发现问题

# 目标

接入的前端项目，提供 7\*24 小时监控任务，第一时间发现页面异常的页面，并给项目负责人发送邮件，报告错误信息

# sentry

专注于开发语言异常信息捕捉，帮助开发及时分析和排查问题

# 优势

- 主流语言支持
  python
  node
  php
  rails
  ruby
  javascript
  ...

- 文档齐全
- 代码
- 实时上报。出错后系统可以实时看到，可以配置邮件通知。
- 用户信息。可以看到报错的IP、浏览器版本、系统版本。
- 调试信息。即使是压缩后的JavaScript文件，上传sourcemap后可以看到源文件报错的行列号，方便调试。


# sentry 要求

- redis
- postgres
- memcached
- smtp

# 不足

数据监控
性能监控
异常监控


sso
邮件 短信
